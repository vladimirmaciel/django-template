# Backend Django

Nesta pasta se econtra os arquivos de backend do projeto Django.

Será usado o Poetry como gestor de pacotes.

Por isso, para instalar o projeto, basta rodar o comando:

```bash
poetry install --sync
```
Para rodar seu servidor Django ative o ambiente virtual com o comando:

```bash
poetry shell
```

Então, rode o servidor com o comando:

```bash
python manage.py runserver
```
