pyenv which python - usado para encontrar o caminho absoluto do executável do Python instalado pelo pyenv

poetry config virtualenvs.in-project true - cria o ambiente virtual na pasta do projeto
poetry config virtualenvs.path .venv - cria o ambiente virtual na pasta .venv

poetry add django - instala o Django
poetry add --dev black - instala o Black
poetry shell - ativa o ambiente virtual criado pelo poetry
