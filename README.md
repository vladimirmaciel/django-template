# django-template
Template de projeto Django, boas prativas 

## Principios
1. A organização do pastas deve ser clara. Seus nomes devem ser auto-explicativos.
2. Cada pasta deve possuir seu arquvio Readme para explicar sua organização e funcionamento.

## Pastas do projeto

Na pasta raiz econtra-se a pasta backend. Dentro dela encontra-se os arquivos respectivos ao backend feito em Django.
